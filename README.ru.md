# Clipboard Manager

**Минималистичная, красивая история буфера обмена для Mac.**

Нативное приложение в менюбаре, которое запоминает всё, что вы копируете.  
Без иконки в Dock, без лишнего — просто иконка клипборда в строке меню.

## Возможности

- **Автоматический захват** — каждый `Cmd+C` сохраняется мгновенно
- **Поиск** — фильтрация сотен элементов за миллисекунды
- **Закрепление** — сохраняйте важные фрагменты навсегда
- **Авто-очистка** — незакреплённые элементы исчезают через час
- **Нативный вид** — вибрация, анимации, тёмная/светлая тема
- **Конфиденциальность** — всё хранится локально

## Использование

Кликните на иконку клипборда в строке меню → откроется история → кликните на любой элемент, чтобы скопировать его обратно.

| Действие | Результат |
|---|---|
| Клик по элементу | Копирование в буфер |
| Иконка булавки | Закрепить элемент навсегда |
| Иконка крестика | Удалить элемент |
| Поле поиска | Фильтрация истории |

## Установка

```bash
git clone https://github.com/sjgagahvabw/clipboard-manager.git
cd clipboard-manager
make run
```

Или откройте проект в Xcode и нажмите Run.

## Сборка

```bash
make        # компиляция + сборка .app
make run    # запуск
make clean  # удаление артефактов сборки
```

Без зависимостей. Никаких пакетных менеджеров. Только Swift.

## Иконка

Приложение находится в строке меню (правый верхний угол).  
Оно **не** отображается в Dock.
